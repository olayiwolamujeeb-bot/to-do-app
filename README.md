# ToDo App

A group project to create a simple ToDoApp, profounding a real time solution to safe track dasily work.

## Features
- Status tracking for tasks
- Task Creation (Dev: Mujeeb)
- Task Completion (Dev: Sobur)
- Task Merge (Dev: Abdulqayyum)

## Project Overview
This project is a lightweight React + Vite application for managing daily tasks. Users can create tasks, update completion status, and merge related items.

## Getting Started
1. Install dependencies:
```bash
npm install
```
2. Run the development server:
```bash
npm run dev
```
3. Open the app in your browser at `http://localhost:5173`.

## Available Scripts
- `npm run dev` - start local development server
- `npm run build` - build production assets
- `npm run preview` - preview built app
- `npm run lint` - lint project files

## Notes
The current project uses React, Vite, Tailwind CSS, and simple client-side routing.
